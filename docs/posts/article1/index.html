8-bit Binary to Integer Converter in C++

March 24, 2024

How to Convert an 8Bit Binary Number into an Integer#

This is a rather simple task to accomplish if you know some basic C++. Firstly, create a file called main.cpp. Then use the code shown below:

#include <cstdio>

int main(){

    //a short is 8 bits, use 0b prefix notation for binary
    unsigned short a = 0b00001011;
    printf("%hu\n", a);
    return 0;
}

To compile and run this on Linux, do this:

g++ main.cpp -o hex_to_int

To run this in Linux do this:

./hex_to_int

Disclaimer: This only work on 8bit integers, since this is meant to be a simple proof of concept. For longer binary numbers, use a different data type to encapsulate your number.
