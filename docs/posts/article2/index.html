Simple HTTP Client - Server for ASCII Data

March 24, 2024

I wrote this back on January 21st 2024 for a coding assessment that I got. The code for the HTTP Client is shown below:

import socket

# Set the server IP address and port
server_ip = 'ip_addr'
server_port = 8080

# Set the path to the text file
file_path = 'proprietary_data.txt'

# Read ASCII data from the text file
with open(file_path, 'r') as file:
    ascii_data = file.read()

# Create a socket object
client_socket = socket.socket(socket.AF_INET, socket.SOCK_STREAM)

# Connect to the server
client_socket.connect((server_ip, server_port))

# Send the ASCII data to the server
client_socket.send(ascii_data.encode('utf-8'))

# Close the socket
client_socket.close()

The code for the HTTP Server is shown below:

import socket

# Set the server IP address and port
server_ip = 'your_server_ip'
server_port = 12345

# Create a socket object
server_socket = socket.socket(socket.AF_INET, socket.SOCK_STREAM)

# Bind the socket to a specific address and port
server_socket.bind((server_ip, server_port))

# Listen for incoming connections (1 connection at a time)
server_socket.listen(1)

print(f"Server listening on {server_ip}:{server_port}")

# Accept a connection from a client
client_socket, client_address = server_socket.accept()
print(f"Connection from {client_address}")

# Receive data from the client
received_data = client_socket.recv(1024).decode('utf-8')

# Save the received data to a file (e.g., received_data.txt)
with open('received_data.txt', 'w') as file:
    file.write(received_data)

# Close the sockets
client_socket.close()
server_socket.close()

In order to use this code, you’ll need to set the IP Address and Port number in both the Client and Server code. Ideally run this on two different machines with a static IP Address to get the best results.
